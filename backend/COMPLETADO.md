## 📞 Ayuda Rápida

### Si ves este error:
```
"Cannot find module"
```
**Solución:**
```bash
rm -rf node_modules
npm install
```

### Si el puerto 3000 está en uso:
**Solución:**
```bash
# Windows
netstat -ano | findstr :3000
taskkill /PID <PID> /F

# O cambiar en vite.config.js
server: { port: 3001 }
```

### Si API no responde:
**Verificar:**
```bash
✓ npm run dev ejecutándose
✓ MongoDB conectado
✓ MONGO_URI correcto en .env
```
