## 🚀 Comienza en 5 Minutos

### Paso 1: Instala el frontend
```bash

npm install
cd ..
```

### Paso 2: Ejecuta ambos servidores
```bash
# Windows
start-all.bat

# Mac/Linux
./start-all.sh
```

### Paso 3: Abre tu navegador
```
http://localhost:3000
```

## 🔌 Integración API

Automática y lista para usar:

```
Frontend ←→ API ←→ MongoDB
  React    Express   CRUD
```

Endpoints disponibles:
- `GET /api/experiencias` - Obtener todas
- `GET /api/experiencias/:id` - Obtener una
- `POST /api/experiencias` - Crear
- `PUT /api/experiencias/:id` - Actualizar
- `DELETE /api/experiencias/:id` - Eliminar
